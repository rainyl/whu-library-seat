    # 申请软件使用权

    WARNING

    不再对桌面版本低于 v1.5.0、安卓版本低于 v1.2.0 的软件提供用户认证支持

    在您登录之前，软件会检查您的学号是否被记录在这个文件中，只有被记录在这个文件中的学号(经过加密)才能使用本软件，如果您的学号(经过加密)没有被记录，您可以通过邮件联系我(请在邮件中注明学号、院系和姓名)

    当然，因为本软件为开源软件，您可以在遵守本软件版权声明的前提下，自己构建软件并删除其中的认证代码，具体请查看面向开发者

    # 软件更新

    • 桌面端 软件启动的时候会自动访问GitHub Release检查更新，如果有更新，在软件的左下角会出现更新的按钮

    • 移动端(Android) 软件启动的时候会自动访问这个文件检查更新，如果有更新，在软件的左下角会出现下载更新的按钮
